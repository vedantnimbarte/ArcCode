# Installation

## Prerequisites

- Rust 1.80 or later (uses the 2021 edition; pinned in `Cargo.toml`).
- A working C toolchain for some transitive crates.
- (Optional) An API key for the provider(s) you intend to use.

## Build from source

```bash
git clone https://github.com/vedantnimbarte/Arc-Code.git
cd Arc-Code
cargo build --release
```

The resulting binary is at `target/release/arccode` (or `arccode.exe` on
Windows).

## Install onto your PATH

```bash
cargo install --path crates/arccode-cli
```

Once installed, run `arccode` from any project directory to launch the
interactive TUI, or `arccode --print "<prompt>"` for a headless one-shot.

## Tree-sitter (optional)

Deep language-aware parsing (Rust, Python, JavaScript, TypeScript, Go) is
feature-gated so the workspace builds without the C toolchain if you don't need
parsing. Enable it when you want semantic chunking in the RAG index, symbol
extraction, AST-aware diffs, and outline generation.
